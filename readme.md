Chromemo
=========

Chromemo is a Memo/Notebook application written for Chrome.


Data Structures
---------------

Memos are stored in an array of Memo objects;

JSON definition of memo object:
```
{title:"Memo title",
 key:"unique key generated from date/time of save",
 data:"memo content"}
 ```



Libraries Used
--------------
* jQuery
* jQueryUI
